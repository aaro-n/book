# 标注导出与迁移方案 — Readium Annotation Set + 多格式导出

> **定位**：定义标注的导入/导出文件格式，支持跨系统迁移和数据备份。
> **参考基准**：Thorium Reader `IReadiumAnnotationSet`（Readium Annotation Set 格式）、Kavita（标注导出）、Polar Bookshelf（JSON 持久化）。
> **更新日期**：2026-07-14

## 一、导出格式

### 1.1 Readium Annotation Set JSON（标准格式）

> 参考：Thorium Reader `IReadiumAnnotationSet` — `@context: "http://www.w3.org/ns/anno.jsonld"`, `type: "AnnotationSet"`

这是**首选导出格式**，可导入其他兼容 Readium 的阅读器。

## 二、Rust 实现

### 2.1 导出 Service

```rust
// src/service/export_service.rs

use crate::models::annotation::Annotation;
use crate::repository::annotation_repo::AnnotationRepo;

pub struct ExportService<R: AnnotationRepo> {
    repo: R,
}

impl<R: AnnotationRepo> ExportService<R> {
    /// 导出为 Readium Annotation Set JSON
    pub async fn export_json(
        &self, doc_id: i32, user_id: i32, doc_meta: &DocumentMeta,
    ) -> Result<AnnotationSet> {
        let annotations = self.repo.find_by_doc(doc_id, user_id).await?;

        Ok(AnnotationSet {
            context: "http://www.w3.org/ns/anno.jsonld".into(),
            set_type: "AnnotationSet".into(),
            about: PublicationMeta {
                identifier: doc_meta.identifier.clone(),
                format: doc_meta.mime_type.clone(),
                title: doc_meta.title.clone(),
                creator: doc_meta.authors.clone(),
                publisher: doc_meta.publisher.clone(),
                date: doc_meta.issued_date.clone(),
            },
            total_items: annotations.len() as u32,
            items: annotations,
        })
    }

    /// 导出为 Markdown
    pub async fn export_markdown(
        &self, doc_id: i32, user_id: i32, doc_meta: &DocumentMeta,
    ) -> Result<String> {
        let annotations = self.repo.find_by_doc(doc_id, user_id).await?;
        let mut md = format!("# {} — 标注导出\n\n", doc_meta.title);
        md.push_str(&format!("> 导出时间：{}\n> 标注总数：{}\n\n---\n\n",
            Utc::now().to_rfc3339(), annotations.len()));

        // 按页码分组
        let by_page = group_by_page(annotations);
        for (page_num, annos) in by_page {
            md.push_str(&format!("## 第 {} 页\n\n", page_num));
            for anno in annos {
                match anno.motivation {
                    AnnotationMotivation::Highlighting => {
                        let text = extract_highlight_text(&anno);
                        md.push_str(&format!("### 🟡 高亮\n> {}\n\n", text));
                    }
                    AnnotationMotivation::Commenting => {
                        let highlight = extract_highlight_text(&anno);
                        let note = extract_note_text(&anno);
                        md.push_str(&format!("### 🟡 高亮\n> {}\n\n", highlight));
                        md.push_str(&format!("📝 **笔记**：{}\n\n---\n\n", note));
                    }
                    AnnotationMotivation::Bookmarking => {
                        md.push_str(&format!("### 🔖 书签\n\n"));
                    }
                    _ => {}
                }
            }
            md.push_str("---\n\n");
        }

        Ok(md)
    }

    /// 导出为 HTML（带 XSS 消毒）
    pub async fn export_html(
        &self, doc_id: i32, user_id: i32, doc_meta: &DocumentMeta,
    ) -> Result<String> {
        let md = self.export_markdown(doc_id, user_id, doc_meta).await?;
        // Markdown → HTML → 消毒
        let raw_html = markdown_to_html(&md);
        let safe_html = ammonia::clean(&raw_html);
        Ok(wrap_html_template(&safe_html, doc_meta))
    }
}
```

### 2.2 导入 Service

```rust
// src/service/import_service.rs

pub struct ImportService<R: AnnotationRepo> {
    repo: R,
}

impl<R: AnnotationRepo> ImportService<R> {
    /// 从 Readium Annotation Set JSON 导入
    pub async fn import_json(
        &self, doc_id: i32, user_id: i32, json: &str,
    ) -> Result<ImportReport> {
        let set: AnnotationSet = serde_json::from_str(json)
            .map_err(|e| AppError::BadRequest(format!("无效的 Annotation Set: {}", e)))?;

        // 验证 @context 和 type
        if set.context != "http://www.w3.org/ns/anno.jsonld" {
            return Err(AppError::BadRequest("无效的 @context".into()));
        }
        if set.set_type != "AnnotationSet" {
            return Err(AppError::BadRequest("无效的 type".into()));
        }

        let mut report = ImportReport::default();

        for anno in set.items {
            // 按 id 去重：已存在的跳过
            if self.repo.exists_by_w3c_id(&anno.id).await? {
                report.skipped.push(anno.id);
                continue;
            }

            // 验证标注
            if let Err(e) = validate_annotation(&anno) {
                report.invalid.push((anno.id.clone(), e.to_string()));
                continue;
            }

            // 写入数据库
            self.repo.create(doc_id, user_id, &anno).await?;
            report.imported.push(anno.id);
        }

        Ok(report)
    }
}

pub struct ImportReport {
    pub imported: Vec<String>,  // 成功导入的标注 ID
    pub skipped: Vec<String>,   // 跳过（已存在）的标注 ID
    pub invalid: Vec<(String, String)>, // 无效标注（ID + 错误原因）
}
```

## 三、导出文件版本化

> **参考**：Polar Bookshelf `DocMeta.version = 1` — 元数据格式版本号。

```json
{
  "@context": "http://www.w3.org/ns/anno.jsonld",
  "type": "AnnotationSet",
  "version": "1.0",
  "exported_by": "book-server v0.1.0",
  "exported_at": "2026-07-14T10:00:00Z",
  "about": { ... },
  "items": [ ... ]
}
```

| 字段 | 说明 |
|------|------|
| `version` | 导出格式版本号，当前 `"1.0"` |
| `exported_by` | 导出软件名称和版本 |
| `exported_at` | 导出时间（ISO 8601 UTC） |

> **向后兼容**：导入时检查 `version` 字段，未来格式变更时可做版本迁移。

## 四、API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| **GET** | `/api/v1/documents/{doc_id}/annotations/export?format=json` | 导出 Readium Annotation Set JSON |
| **GET** | `/api/v1/documents/{doc_id}/annotations/export?format=markdown` | 导出 Markdown |
| **GET** | `/api/v1/documents/{doc_id}/annotations/export?format=html` | 导出 HTML |
| **POST** | `/api/v1/documents/{doc_id}/annotations/import` | 导入 Annotation Set JSON |

> **Content-Disposition**：导出时设置 `Content-Disposition: attachment; filename="书名-标注.json"` 触发浏览器下载。
